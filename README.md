## Patch for Xiaomi MIUI China

中国版 MIUI に日本語翻訳を追加する Magisk モジュールです。

magisk-module-miui-cn2global.zip をダウンロードして Magisk アプリのモジュールタブからこのモジュールをインストール・再起動することで、言語設定で日本語が選べるようになります。

MIUI 13 にて動作確認しています。
